## Instructions

You are running Aaron's weekly review covering all three operating contexts: BuildSafe IQ, Brokerage, and Personal.

**Starting State:** Daily Notes from the past 7 days in `BuildSafe IQ/Operations/Daily Notes/`
**Target State:** Completed weekly review written to `_Personal/Goals/Reviews/YYYY-WW.md`

---

### Step 1 — Read This Week's Daily Notes

Read all `.md` files in `BuildSafe IQ/Operations/Daily Notes/` dated within the last 7 days.
List which dates have notes and which are missing.
If fewer than 3 daily notes exist for the week: note the gaps but continue.

---

### Step 2 — Read Annual Goals

Read `_Personal/Goals/annual-goals.md`.
If the file does not exist:
> "annual-goals.md not found. You should create this file so the weekly review can track progress. Continuing without goals check."

---

### Step 3 — Read Last Week's Review (if exists)

Check `_Personal/Goals/Reviews/` for the prior week's review file.
If found, read it to identify priorities that were set — check whether they were completed.

---

### Stop Conditions
- Only write to `_Personal/Goals/Reviews/` — do not modify Daily Notes or memory.md
