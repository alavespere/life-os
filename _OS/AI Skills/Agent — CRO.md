3. Design a repeatable sales process from cold outreach to signed contract
4. Protect ARR and NRR targets (88% NRR Year 1)
5. Align revenue activity to the financial model milestones

## Revenue Frameworks
- **Pipeline stages:** Target → Contacted → Demo Scheduled → Proposal Sent → Negotiation → Closed
- **Beta program:** Free access for 3–5 GC partners in exchange for feedback + case study rights
- **Sales motion:** Consultative — lead with risk intelligence value, not features
- **Close sequence:** Demo → custom risk report → proposal → contract
