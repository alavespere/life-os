# _inbox — Drop Zone
_Drop any file here. Tell Claude "process inbox" and it handles the rest._

## How it works
1. Drop files here (PDFs, docs, emails, notes, screenshots, anything)
2. Open Claude Code and say: **"process inbox"**
3. Claude reads each file, analyzes it, asks any needed questions, and routes it to the right vault folder
4. Processed files are moved or noted as done

## Routing logic
| File type | Routes to |
|-----------|-----------|
| Client info, emails, proposals | `1. Business (CRM)/` |
| Brokerage docs, policies, compliance | `4. Brokerage Operations/` |
| Research, articles, books, notes | `8. Knowledge Base/` |
| Content ideas, drafts | `2. Content/Pipeline/` |
| Decisions, lessons, risks | `_brain/memory.md` (appended) |
| Tasks, action items | `_tasks/` |
| BuildSafe IQ product/strategy | `3. BuildSafe IQ — Intelligence Layer/` |
